## Structure

[src](/src) - Source folder of application
- [app](/src/app) - Routing with pages
- [widgets](/src/widgets) - Large blocks with large business logic & UI
- [features](/src/features) - One feature is used only for one business logic
- [entities](/src/entities) - Entities of projects
- [shared](/src/shared) - Reusable entities
    - [api](/src/shared/api) - API call system
    - [types](/src/shared/types) - Reusable types
    - [ui](/src/shared/ui) - UI components 
    - [libs](/src/shared/libs) - Libraries for APIs
